# stunnel

[![Build](https://github.com/gsmlg-ci/stunnel/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/stunnel/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/stunnel`
- `ghcr.io/gsmlg-dev/stunnel`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/stunnel:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/stunnel:latest
```

## Build

```bash
docker build -t gsmlg/stunnel .
```
